Break → Force stop command Ctrl + M → Enter Mark Mode Ctrl + V → Paste text (or Right-click) Ctrl + ↑ / ↓ → Scroll line by line Ctrl + PgUp/PgDn→ Scroll page by page Ctrl + F → Find text Ctrl + Home/End → Jump to top/bottom Text Selection & Editing ------------------------ Shift + Arrows → Select text Shift + Home / End → Select to start/end of line Shift + PgUp / PgDn → Select page up/down Ctrl + Shift + Arrows → Select word by word Enter → Copy selected text Right-click → Paste clipboard Command History --------------- ↑ / ↓ Arrows → Cycle through previous commands F7 → Show command history Alt + F7 → Clear history F8 → Search history F9 → Run command by number Function Keys ------------- F1 → Repeat last command char by char F2 → Copy up to a character F3 → Repeat last command F4 → Delete up to a character F5 → Repeat last command (no cycling) F6 → Insert ^Z (EOF) F10 → Repeat last command Window Control -------------- Alt + Enter → Toggle full screen Ctrl + Shift + Enter → Run CMD as admin Ctrl + Tab → Switch CMD windows CMD Commands Cheat Sheet ======================== Navigation ---------- cd [folder] → Change directory cd .. → Go up one directory dir → List files and folders tree → Display folder structure File & Folder Management ------------------------ mkdir [folder] → Create a new folder rmdir [folder] → Remove a folder (empty only) rmdir /s [folder] → Remove a folder and all contents del [file] → Delete a file copy [src] [dest] → Copy file(s) move [src] [dest] → Move file(s) ren [old] [new] → Rename a file System Info ----------- cls → Clear the screen echo [text] → Display text systeminfo → Show system details ver → Show Windows version tasklist → List running processes taskkill /IM [exe] /F → Kill a process Networking ---------- ipconfig → Show IP configuration ipconfig /all → Detailed network info ping [host] → Test connection to host tracert [host] → Trace route to host netstat → Show active connections nslookup [host] → DNS lookup Utilities --------- help → Show available commands help [command] → Show help for a command exit → Close CMD
